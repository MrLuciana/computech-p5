ตัวแปร

ตัวแปรคืออะไร?

ตัวแปรคือชื่อที่ใช้เก็บข้อมูลในโปรแกรม สามารถเปลี่ยนแปลงค่าได้

# การประกาศตัวแปร
name = "สมชาย"
age = 12
score = 85.5
is_student = True

กฎการตั้งชื่อตัวแปร

สิ่งที่ทำได้
  • ขึ้นต้นด้วยตัวอักษรหรือ _
  • ใช้ตัวอักษร ตัวเลข และ _
  • ชื่อสื่อความหมาย เช่น student_name
สิ่งที่ควรหลีกเลี่ยง
  • ใช้คำสงวน เช่น if, for
  • ขึ้นต้นด้วยตัวเลข
  • ใช้สัญลักษณ์พิเศษ เช่น @, #

ประเภทข้อมูลพื้นฐาน

ประเภท ตัวอย่าง คำอธิบาย
Integer 10, -5, 0 จำนวนเต็ม
Float 3.14, -0.5 จำนวนทศนิยม
String "Hello", 'สวัสดี' ข้อความ
Boolean True, False ค่าความจริง
