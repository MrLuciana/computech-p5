ผังงาน

การเขียนโปรแกรมคอมพิวเตอร์

โครงสร้างพื้นฐาน
  • ส่วนประกาศตัวแปร
  • ส่วนรับข้อมูล
  • ส่วนประมวลผล
  • ส่วนแสดงผล
ตัวอย่างโค้ด
print("Hello, World!")
name = input("ชื่อของคุณคืออะไร? ")
print("สวัสดี", name)

การตรวจสอบข้อผิดพลาดของโปรแกรม

ข้อผิดพลาดทั่วไป
  • Syntax Error (ไวยากรณ์ผิด)
  • Logic Error (ตรรกะผิด)
  • Runtime Error (เกิดขณะทำงาน)
วิธีแก้ไข
  • ตรวจสอบบรรทัดที่แจ้ง error
  • ทดสอบทีละส่วน
  • ใช้เครื่องมือ debug

การเขียนโปรแกรมด้วย Scratch

ส่วนประกอบสำคัญ
  • บล็อกคำสั่ง
  • สไปรท์ (ตัวละคร)
  • ฉากหลัง
ไปที่เว็บ Scratch

ตัวแปร

ความหมาย

ตัวแปรคือชื่อที่ใช้เก็บข้อมูลในโปรแกรม

เช่น score = 100

ข้อควรระวัง
  • ตั้งชื่อให้สื่อความหมาย
  • ระวังการใช้ตัวอักษรใหญ่เล็ก
  • ไม่ใช้คำสงวน

การเขียนโปรแกรมตรวจสอบเลขคู่และเลขคี่

ตัวอย่างโค้ด
number = int(input("ป้อนตัวเลข: "))
if number % 2 == 0:
    print("เลขคู่")
else:
    print("เลขคี่")
ผังงาน

การคำนวณความสมส่วนของร่างกาย

สูตรคำนวณ BMI

น้ำหนัก (kg) / ส่วนสูง2 (m)

ตัวอย่างโปรแกรม 
weight = float(input("น้ำหนัก (kg): "))
height = float(input("ส่วนสูง (m): "))
bmi = weight / (height ** 2)
print("ค่า BMI ของคุณคือ:", round(bmi, 2))
ตารางค่า BMI
ค่า BMI ความหมาย
น้อยกว่า 18.5 น้ำหนักน้อย
18.5 - 22.9 ปกติ
23.0 - 24.9 ท้วม
25.0 ขึ้นไป อ้วน
