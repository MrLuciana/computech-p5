การเขียนโปรแกรม

โครงสร้างพื้นฐานของโปรแกรม

1. การรับข้อมูล

รับข้อมูลจากผู้ใช้หรือแหล่งข้อมูลอื่นๆ

input("ป้อนชื่อของคุณ: ")
2. การประมวลผล

คำนวณหรือประมวลผลข้อมูล

result = number1 + number2
3. การแสดงผล

แสดงผลลัพธ์ให้ผู้ใช้

print("ผลลัพธ์คือ:", result)

ตัวอย่างโปรแกรมอย่างง่าย

โปรแกรมคำนวณพื้นที่สี่เหลี่ยม 
# รับค่าความยาวและความกว้าง
length = float(input("ป้อนความยาว: "))
width = float(input("ป้อนความกว้าง: "))

# คำนวณพื้นที่
area = length * width

# แสดงผลลัพธ์
print("พื้นที่สี่เหลี่ยมคือ:", area)
