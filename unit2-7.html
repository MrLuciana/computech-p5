เลขคู่/คี่

การตรวจสอบเลขคู่และเลขคี่

เลขคู่คือจำนวนเต็มที่หารด้วย 2 ลงตัว ส่วนเลขคี่คือจำนวนเต็มที่หารด้วย 2 ไม่ลงตัว

ผังงาน
โค้ดตัวอย่าง 
number = int(input("ป้อนตัวเลข: "))

if number % 2 == 0:
    print("เลขคู่")
else:
    print("เลขคี่")

การใช้งานเงื่อนไข if-else

คำอธิบาย ตัวอย่างโค้ด
เงื่อนไข if if condition:
  # ทำเมื่อเงื่อนไขเป็นจริง
เงื่อนไข if-else if condition:
  # ทำเมื่อเงื่อนไขเป็นจริง
else:
  # ทำเมื่อเงื่อนไขเป็นเท็จ
ตัวดำเนินการ % (modulo) remainder = 7 % 3 # ได้ 1
