คำนวณความสมส่วน (BMI)

สูตรคำนวณ BMI

BMI = น้ำหนัก (kg) / (ส่วนสูง (m) × ส่วนสูง (m))

ตัวอย่างโปรแกรม 
weight = float(input("น้ำหนัก (kg): "))
height = float(input("ส่วนสูง (m): "))

bmi = weight / (height ** 2)

print("ค่า BMI ของคุณคือ:", round(bmi, 2))
ตารางค่า BMI
ค่า BMI ความหมาย
น้อยกว่า 18.5 น้ำหนักน้อย
18.5 - 22.9 ปกติ
23.0 - 24.9 ท้วม
25.0 ขึ้นไป อ้วน

การประยุกต์ใช้เงื่อนไข

โปรแกรมประเมินผล BMI 
if bmi < 18.5:
    print("น้ำหนักน้อย")
elif 18.5 <= bmi < 23:
    print("ปกติ")
elif 23 <= bmi < 25:
    print("ท้วม")
else:
    print("อ้วน")
